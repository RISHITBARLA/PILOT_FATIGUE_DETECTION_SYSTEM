# PILOT_FATIGUE_DETECTION_SYSTEM
Pilots don't get second chances. I built a real-time fatigue detection system using computer vision and deep learning that monitors eye blink rate, head pose, and facial landmarks — and alerts before drowsiness becomes dangerous. Built with Python, OpenCV &amp; TensorFlow. Tested live. It works.
